# ICP builder

Build a practical Ideal Customer Profile with deterministic outputs for targeting, scoring, and execution planning. This tool runs entirely in the browser (Alpine.js).

## Live tool

**[https://urlcv.com/tools/icp-builder](https://urlcv.com/tools/icp-builder)**

## Features

- Guided ICP wizard (industry, size, geography, personas, triggers, and rules)
- Must-have vs nice-to-have criteria with positive and negative targeting filters
- Weighted scoring model and account-level 0-100 fit scoring
- Trigger prioritisation matrix and title include/exclude builder
- Deterministic LinkedIn/Sales Nav/Boolean query blocks
- Messaging angle matrix and 30-day outbound test planner
- TAM-lite estimator, validation guardrails, and confidence score
- Copyable ICP one-pager, print view, CSV starter schema, and local version snapshots
